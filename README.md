# Pulse

**Ecommerce Intelligence Platform**

Pulse is a real-time analytics and predictive intelligence platform for ecommerce businesses. Built by Echoe Labs.

## Features

- **Real-Time Tracking** - Monitor every click, view, and purchase as it happens
- **Predictive Analytics** - ML-powered forecasting for inventory, revenue, and customer behavior
- **Instant Insights** - Auto-generated reports and alerts when metrics change
- **Universal Integration** - Connect to Shopify, WooCommerce, custom stores, and 100+ platforms

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **3D/WebGL**: Three.js, React Three Fiber
- **Animation**: Framer Motion
- **Smooth Scrolling**: Lenis
- **UI Components**: Shadcn UI

## Getting Started

Install dependencies:

```bash
npm install
# or
bun install
```

Run the development server:

```bash
npm run dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```
